Nutriplan is a webapp designed to gather search results using Spoonacular's Api, and allow the user to save those results for later.

--For Developers--
getJSONData.php is the script for retrieving data from the database.

putJSONData.php is the script for saving data to the database.

gridWithModal(Spoon).html is the main home page of our webapp.

results.html is the page the shows search results

saved.html shows a list of all of your saved recipes

savedRecipe.html shows an individual recipe that you have saved.


The files in the Interface folder were mostly used to draft out the interface for Nutriplan, and mostly go unused in the current build.
